# Gaithubs Website

موقع احترافي متعدد الصفحات لعرض خدمات التصميم ثلاثي الأبعاد وBlender.

## التشغيل

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```

## الصفحات
- الرئيسية
- المشاريع
- الخدمات
- من نحن
- تحدث معنا

## الرفع إلى GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPOSITORY.git
git push -u origin main
```
